# Billboard Management System

## Overview

Billboard Management System is a Java JSP and Servlet based web application developed to manage billboard advertising operations.

The system allows administrators to manage billboard locations, billboard sizes, customers, purchases, payments, inspectors, and work allocations efficiently.

## Features

* User Login Management
* Location Management
* Billboard Size Management
* Location and Size Mapping
* Customer Management
* Billboard Purchase Management
* Payment Tracking
* Inspector Management
* Work Allocation Management
* Status Monitoring

## Technologies Used

### Frontend

* HTML
* JavaScript
* JSP

### Backend

* Java
* Servlets
* JDBC

### Database

* MySQL

### Server

* Apache Tomcat

## Database Tables

* login
* location
* billboard_size
* location_size
* customer
* billboard_purchase
* inspector
* work_allocation
* payment

## Project Workflow

1. Admin Login
2. Add Locations
3. Add Billboard Sizes
4. Configure Location Size Pricing
5. Register Customers
6. Create Billboard Purchases
7. Record Payments
8. Assign Work to Inspectors
9. Track Work Status

## Setup Instructions

1. Clone the repository

```bash
git clone https://github.com/AshikMohammad-dev/Billboard-Management-System.git
```

2. Import the project into Eclipse

3. Configure Apache Tomcat Server

4. Import the database using:

```sql
billboard_db.sql
```

5. Update database credentials in the connection class

6. Run the project on Tomcat
